Quick Start
1. Get an API Key

Contact the administrator to get your API key.

2. Make API Requests
curl -H "X-API-Key: your_api_key" \
  "{{ request.host_url }}api/v1/video?url=https://youtube.com/watch?v=VIDEO_ID"
3. Stream Content
{{ request.host_url }}stream/video/VIDEO_ID?api_key=your_api_key
Shared Caching: Once any user requests content, it's cached for all users with valid API keys!
